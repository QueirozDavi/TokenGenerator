# Token Generator

### What is this repository for? ###

* MicroService who generate tokens and sent they by email, to be validated.
* Version (1.0)

### Technologies? ###

* Java 8
* Spring Boot 2.1.9.RELEASE
* Jackson
* Swagger
* Lombok
* MongoDB
* Shedlock
